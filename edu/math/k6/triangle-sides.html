📖 一步一步学——为什么三角形有"三边关系"？
1准备开始
三角形是由三条线段首尾相连围成的封闭图形。但三根任意长度的线都能围起来吗？我们来试试看。
🌟 你知道吗？
三角形三边关系定理：三角形任意两边之和大于第三边。
如果 a + b ≤ c 或者 a + c ≤ b 或者 b + c ≤ a，那这三条线段就 不能 围成三角形！

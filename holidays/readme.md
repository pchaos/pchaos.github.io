电子烟花秀 (Electronic Fireworks Show)

一个基于HTML5 Canvas的交互式动画页面，可自定义显示文字和卡通小人数量，支持1-10个小人，自动在烟花和小人状态间切换，适合节日庆祝、活动展示等场景。

功能特点

• ✨ 循环播放烟花效果和卡通小人图片（每10秒自动切换）
• 🎯 通过URL参数自定义显示文字（支持Base64编码）
• 👥 控制小人数量（1-10个），支持男孩女孩混合显示
• 📱 响应式设计，适配各种屏幕尺寸
• 🎨 鲜艳的色彩和流畅的动画效果
• 🔗 一键生成分享链接（支持带控制面板和不带控制面板两种链接）
• 🎭 智能缩放：小人数量超过6个时，中间两个显示为"成人"，其他显示为小孩
• 🖱️ 多种控制面板控制方式：左上角按钮、面板关闭按钮、左下角隐形切换器

快速开始

基本使用

1. 下载所有文件到同一目录
2. 在浏览器中打开 greetings.html（主入口文件）
3. 页面将自动开始播放动画（烟花 ↔ 小人，每10秒切换）

使用控制面板

1. 点击左上角"控制面板"按钮（或左下角隐形区域）
2. 输入要显示的文字（默认：2026 Happy!）
3. 选择小人个数（1-10）
4. 点击"生成Base64链接"按钮生成带控制面板的链接
5. 点击"生成预览链接"按钮生成不带控制面板的纯预览链接
6. 复制生成的链接分享给他人

控制参数使用

URL参数格式

greetings.html?text=Base64编码的文字&count=Base64编码的个数&showPanel=true/false

参数说明

参数 类型 说明 示例

text 字符串 要显示的文字（Base64编码，URL安全格式） text=MjAyNiBIYXBweSE=

count 数字 小人个数（Base64编码，1-10） count=Mg==

showPanel 布尔值 是否显示控制面板 showPanel=true

使用示例

节日祝福：

greetings.html?text=5paw5rGf5LqU5Y2B&count=NA&showPanel=false

• 显示："新年快乐"
• 4个小人
• 隐藏控制面板

生日祝福：

greetings.html?text=55Sf5rS75LqG5LiN5Y+v&count=Mg&showPanel=true

• 显示："生日快乐"
• 2个小人
• 显示控制面板

Base64编码方法

项目中已经内置了URL安全的Base64编码函数，位于main.js中：

```javascript
function base64Encode(str) {
  return btoa(encodeURIComponent(str))
    .replace(/\+/g, "-")
    .replace(/\//g, "_")
    .replace(/=+$/, "");
}
```

使用示例：

```javascript
const text = "2026 Happy!";
const encodedText = base64Encode(text); // MjAyNiBIYXBweSE=
```

在线工具

也可以使用在线Base64编码工具，但需要注意：
1. 先使用UTF-8编码
2. 再进行Base64编码
3. 最后转换为URL安全格式（将+替换为-，/替换为_，去掉末尾=）

常用在线工具：
- https://www.base64encode.org/
- https://www.base64decode.org/

注意：在线工具通常需要手动进行URL安全转换。

技术特性

动画效果

• 烟花粒子系统：使用Canvas实现高性能的粒子动画，支持随机颜色和重力效果
• 星星背景：动态闪烁的星空效果，每个星星有独立的闪烁频率
• 小人动画：SVG矢量图形，男孩女孩交替显示，支持浮动动画效果
• 背景过渡：12种鲜艳背景颜色循环切换，每10秒自动变化
• 智能缩放：小人数量超过6个时，中间两个显示为"成人"，其他显示为小孩

响应式设计

• 自动适应不同屏幕尺寸
• 移动设备优化显示
• 文字大小自适应（根据文字长度自动调整字体大小）
• 画布尺寸随窗口大小自动调整

交互功能

• 多种控制面板控制方式：
  - 左上角"控制面板"按钮
  - 面板内的关闭按钮
  - 左下角10x10px隐形切换区域
• 一键生成两种分享链接：
  - 带控制面板的完整版本链接
  - 不带控制面板的纯预览链接
• 链接复制到剪贴板
• 状态提示消息（成功/错误，3秒后自动隐藏）
• 实时预览：输入文字和选择人数后，页面立即更新显示效果
• 键盘支持：输入框支持回车键触发链接生成

注意事项

1. 本地文件限制：某些浏览器可能阻止本地文件运行JavaScript，建议在Web服务器上运行
2. 复制功能：需要浏览器支持Clipboard API
3. 编码格式：确保使用正确的URL安全Base64编码
4. 文字长度：过长的文字可能影响显示效果，建议不超过20个字符

故障排除

常见问题

问题1：页面显示空白
• 解决方案：检查浏览器控制台错误信息，确保所有文件在同一目录，确认打开的是greetings.html文件

问题2：文字显示乱码
• 解决方案：检查Base64编码是否正确，确保使用URL安全编码（+替换为-，/替换为_，去掉末尾=）

问题3：控制面板不显示
• 解决方案：URL中添加 &showPanel=true 参数，或点击左下角隐形区域（10x10px）

问题4：动画卡顿
• 解决方案：关闭其他占用资源的程序，降低小人数量，或使用简化版本（greeting.html）

问题5：链接复制失败
• 解决方案：确保浏览器支持Clipboard API，或手动复制链接地址

问题6：小人显示异常
• 解决方案：检查kidsCount参数是否在1-10范围内，刷新页面重新加载

调试技巧

1. 按F12打开开发者工具查看错误信息
2. 检查网络面板确保所有文件加载成功
3. 使用控制台测试Base64编码函数：在控制台输入 base64Encode("测试文字")
4. 验证URL参数格式是否正确：检查text和count参数的值
5. 检查控制台日志：页面加载时会输出当前显示文字和状态信息
6. 测试不同小人数量：观察超过6个小人时的"成人"显示效果
